# Database Design (Core Tables)

### Users

```sql
users
```

| Field    | Type    |
| -------- | ------- |
| id       | UUID    |
| name     | VARCHAR |
| email    | VARCHAR |
| phone    | VARCHAR |
| password | VARCHAR |

---

### Student Profile

```sql
student_profile
```

| Field              | Type    |
| ------------------ | ------- |
| user_id            | FK      |
| tenth_percentage   | FLOAT   |
| twelfth_percentage | FLOAT   |
| exam_type          | VARCHAR |
| percentile         | FLOAT   |
| rank               | INT     |
| city               | VARCHAR |
| branch             | VARCHAR |

---

### Colleges

```sql
colleges
```

| Field        | Type    |
| ------------ | ------- |
| college_id   | UUID    |
| college_name | VARCHAR |
| city         | VARCHAR |
| branch       | VARCHAR |
| fees         | FLOAT   |
| avg_package  | FLOAT   |
| scholarship  | BOOLEAN |
