# College Buddy AI V4

College Buddy AI is a Flask-based web application that helps students make informed academic decisions.

## Features

* Student Login & Registration
* College Predictor
* Safe / Target / Dream College Analysis
* Scholarship Predictor
* Placement Predictor
* Prediction History
* PDF Report Generation
* SQLite Database Integration

## Technologies Used

* Python
* Flask
* Pandas
* SQLite
* HTML
* CSS
* ReportLab

## Installation

1. Install dependencies:

pip install -r requirements.txt

2. Run the application:

python app.py

3. Open browser:

http://127.0.0.1:5000
